JavaScript 基础练习

1. 变量定义

知识点：JavaScript 中使用 var、let、const 定义变量

  • var：函数作用域，可重复声明，存在变量提升
  • let：块级作用域，不可重复声明
  • const：块级作用域，定义常量，初始化后不可重新赋值

3. 表达式

知识点：算术、比较、逻辑、条件表达式

5. 函数

知识点：函数的定义和调用方式

  • 函数声明：function name() {}
  • 函数表达式：const name = function() {}
  • 箭头函数：const name = () => {}
  • 参数默认值：function(a = 1) {}
  • rest 参数：function(...args) {}
